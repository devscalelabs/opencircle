The Lessons feature provides individual learning modules within courses in OpenCircle.

## Overview

Lessons deliver focused content through:
- Structured learning modules
- Multi-format content support
- Interactive elements
- Assessment tools
- Progress tracking

## Key Components

- Lesson creation and editing
- Content embedding (videos, text, quizzes)
- Navigation controls
- Progress indicators
- Completion tracking

## Usage

Lessons form the building blocks of courses, allowing educators to create granular learning experiences and enabling learners to progress through structured content.
