The Dashboard provides administrative overview and analytics for OpenCircle.

## Overview

The admin dashboard includes:
- System statistics and metrics
- User activity monitoring
- Content analytics
- Performance indicators
- Administrative alerts

## Key Components

- Analytics widgets
- Statistics displays
- Activity monitors
- Performance metrics
- Alert systems

## Usage

Administrators can monitor the health and activity of the platform, track key metrics, and access important administrative functions from a centralized dashboard.
