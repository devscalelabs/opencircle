# Troubleshooting

Solutions to common issues you might encounter while using OpenCircle.

## Login and Account Issues

### Cannot Log In

**Symptoms:** Login fails, error messages, or redirects back to login page.

**Solutions:**
1. **Verify credentials** - Double-check username and password
2. **Reset password** - Use the "Forgot password?" link
3. **Clear browser cache** - Remove cached data
4. **Try different browser** - Test in another browser
5. **Check account status** - Ensure account isn't deactivated
6. **Contact admin** - Your account may need reactivation

### Email Verification Not Received

**Symptoms:** Verification email doesn't arrive after registration.

**Solutions:**
1. **Check spam folder** - Look in junk/spam
2. **Wait a few minutes** - Email may be delayed
3. **Resend verification** - Use resend option if available
4. **Check email address** - Verify correct email was used
5. **Whitelist sender** - Add to safe senders list
6. **Contact admin** - Request manual verification

### Password Reset Not Working

**Symptoms:** Reset link doesn't work or email not received.

**Solutions:**
1. **Check spam folder** - Look for reset email
2. **Use correct email** - Verify email address
3. **Check link expiration** - Links expire after time
4. **Request new link** - Try password reset again
5. **Clear browser cache** - Remove old data
6. **Contact admin** - Request manual password reset

## Content Issues

### Posts Not Appearing

**Symptoms:** Posted content doesn't show up on timeline.

**Solutions:**
1. **Refresh the page** - Reload to see new content
2. **Check filters** - Verify no filters are hiding content
3. **Verify post was created** - Check your profile
4. **Check channel** - Ensure you're viewing correct channel
5. **Wait a moment** - Content may take time to appear
6. **Check for errors** - Look for error messages

### Cannot Upload Images/Videos

**Symptoms:** Media upload fails or shows errors.

**Solutions:**
1. **Check file size** - Ensure within limits (usually 5-10MB)
2. **Verify file format** - Use supported formats (JPG, PNG, GIF, MP4)
3. **Compress file** - Reduce file size
4. **Check internet** - Ensure stable connection
5. **Try different file** - Test with another image
6. **Clear browser cache** - Remove cached data
7. **Try different browser** - Test in another browser

### Cannot Edit or Delete Posts

**Symptoms:** Edit/delete options not available or not working.

**Solutions:**
1. **Verify ownership** - You can only edit your own posts
2. **Check permissions** - Ensure you have edit rights
3. **Refresh page** - Reload and try again
4. **Check time limits** - Some instances limit edit time
5. **Clear cache** - Remove browser cache
6. **Contact admin** - May need admin assistance

### Comments Not Posting

**Symptoms:** Comments fail to submit or disappear.

**Solutions:**
1. **Check internet connection** - Ensure stable connection
2. **Verify comment isn't empty** - Add content
3. **Check character limits** - Stay within limits
4. **Refresh page** - Reload and try again
5. **Clear browser cache** - Remove cached data
6. **Try different browser** - Test in another browser

## Course Issues

### Cannot Enroll in Course

**Symptoms:** Enrollment button doesn't work or shows error.

**Solutions:**
1. **Check if already enrolled** - View your enrolled courses
2. **Verify course is published** - Course must be active
3. **Check prerequisites** - May require other courses first
4. **Refresh page** - Reload and try again
5. **Clear cache** - Remove browser cache
6. **Contact admin** - May need manual enrollment

### Lessons Not Loading

**Symptoms:** Lesson content doesn't display or videos won't play.

**Solutions:**
1. **Check internet speed** - Ensure adequate bandwidth
2. **Refresh page** - Reload the lesson
3. **Clear browser cache** - Remove cached data
4. **Update browser** - Use latest version
5. **Disable extensions** - Turn off ad blockers
6. **Try different browser** - Test in another browser
7. **Check video format** - Ensure browser supports format

### Progress Not Saving

**Symptoms:** Lesson completion doesn't save or resets.

**Solutions:**
1. **Click "Mark Complete"** - Manually mark if needed
2. **Check internet** - Ensure stable connection
3. **Wait before leaving** - Let progress save
4. **Refresh and check** - Reload to verify
5. **Try again** - Re-complete the lesson
6. **Contact admin** - Report persistent issues

## Notification Issues

### Not Receiving Notifications

**Symptoms:** No notifications for mentions, reactions, etc.

**Solutions:**
1. **Check notification settings** - Ensure enabled
2. **Grant browser permissions** - Allow notifications
3. **Check email settings** - For email notifications
4. **Verify you're logged in** - Must be authenticated
5. **Refresh page** - Reload to sync
6. **Clear cache** - Remove browser cache
7. **Check spam** - For email notifications

### Too Many Notifications

**Symptoms:** Overwhelmed with notification alerts.

**Solutions:**
1. **Adjust settings** - Disable non-essential types
2. **Use digest mode** - Batch notifications
3. **Mute channels** - Reduce channel notifications
4. **Unfollow** - Stop following busy users/topics
5. **Mark all read** - Clear notification backlog

## Browser-Specific Issues

### Chrome Issues

**Solutions:**
1. Clear cache: Settings > Privacy > Clear browsing data
2. Disable extensions: Settings > Extensions
3. Update Chrome: Settings > About Chrome
4. Reset settings: Settings > Advanced > Reset

### Firefox Issues

**Solutions:**
1. Clear cache: Options > Privacy > Clear Data
2. Disable add-ons: Add-ons > Extensions
3. Update Firefox: Menu > Help > About Firefox
4. Refresh Firefox: Help > Troubleshooting Information

### Safari Issues

**Solutions:**
1. Clear cache: Safari > Clear History
2. Disable extensions: Safari > Preferences > Extensions
3. Update Safari: System Preferences > Software Update
4. Reset Safari: Safari > Reset Safari

## Error Messages

### "Network Error"

**Meaning:** Connection to server failed.

**Solutions:**
1. Check internet connection
2. Refresh the page
3. Wait and try again
4. Check if server is down
5. Contact administrator

### "Permission Denied"

**Meaning:** You don't have access to perform action.

**Solutions:**
1. Verify you're logged in
2. Check your user role
3. Ensure you have permission
4. Contact administrator

### "Not Found" or 404

**Meaning:** Content or page doesn't exist.

**Solutions:**
1. Check URL is correct
2. Content may have been deleted
3. Try searching for content
4. Go back to homepage
5. Contact administrator

## Getting Further Help

If these solutions don't resolve your issue:

1. **Check [FAQ](/docs/resources/faq)** - Common questions
2. **Review documentation** - Full guides available
3. **Ask community** - Post in support channels
4. **Contact admin** - Reach out to instance administrator
5. **Report bug** - Submit on [GitHub Issues](https://github.com/devscalelabs/opencircle/issues)

## Reporting Issues

When reporting issues, include:
- **Description** - What's happening
- **Steps to reproduce** - How to recreate issue
- **Expected behavior** - What should happen
- **Actual behavior** - What actually happens
- **Browser/device** - Your setup
- **Screenshots** - Visual evidence if possible
- **Error messages** - Any errors shown
