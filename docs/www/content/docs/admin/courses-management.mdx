# Course Management

Comprehensive guide to creating, managing, and administering courses on OpenCircle.

## Creating Courses

### Course Creation Process

1. Navigate to **Courses** in admin dashboard
2. Click **Create Course** or **New Course**
3. Fill in course details:
   - Title
   - Description
   - Cover image
   - Category/tags
4. Save course

## Managing Lessons

### Creating Video Lessons

1. Select section
2. Click **Add Lesson** > **Video**
3. Upload video or add URL
4. Set lesson title and description
5. Save lesson

### Creating Text Lessons

1. Select section
2. Click **Add Lesson** > **Text**
3. Use rich text editor
4. Add content, images, code blocks
5. Save lesson

### Creating Quizzes

1. Select section
2. Click **Add Lesson** > **Quiz**
3. Add questions
4. Set correct answers
5. Configure passing score
6. Save quiz

### Creating Assignments

1. Select section
2. Click **Add Lesson** > **Assignment**
3. Add instructions
4. Set submission requirements
5. Configure grading (if supported)
6. Save assignment

## Course Settings

### Course Configuration

Configure course options:
- **Visibility** - Public or private
- **Enrollment** - Open or restricted
- **Pricing** - Free or paid (if supported)
- **Certificate** - Enable completion certificates
- **Prerequisites** - Required courses

### Publishing Courses

Publish your course:
1. Complete all sections and lessons
2. Review course content
3. Set course as published
4. Course becomes available to users
