# Platform Settings

Configure and customize your OpenCircle instance through the platform settings.

## Accessing Settings

Navigate to **Settings** or **App Settings** in the admin dashboard to access platform configuration.

**IMAGE PLACEHOLDER: Settings page**
*Description: Screenshot of the platform settings interface*

## General Settings

### Platform Information

Configure basic platform details:
- **Platform Name** - Your instance name
- **Platform Description** - Brief description
- **Platform URL** - Your domain
- **Contact Email** - Support email

**IMAGE PLACEHOLDER: General settings**
*Description: Screenshot of general settings section*

## Registration Settings

### Sign-Up Configuration

Control user registration:
- **Enable Sign-Up** - Allow new registrations
- **Require Email Verification** - Verify email addresses
- **Require Invite Code** - Make invite codes mandatory
- **Default User Role** - Role for new users

### Login Options

Configure authentication:
- **Email/Password** - Standard login
- **GitHub OAuth** - GitHub authentication
- **Other OAuth** - Additional providers (if supported)
- **Password Requirements** - Minimum length, complexity

## Content Settings

### Content Configuration

Manage content features:
- **Enable Articles** - Allow article creation
- **Enable Courses** - Enable course features
- **Enable Resources** - Allow resource sharing
- **Content Moderation** - Require approval
- **Max Upload Size** - File size limits

- **Storage Provider** - Cloudflare R2, S3, etc.
- **Max File Size** - Upload limits
- **Allowed Formats** - Supported file types
- **Image Compression** - Optimize images

## Notification Settings

### Platform Notifications

Configure notification system:
- **Email Notifications** - Enable/disable
- **Push Notifications** - Browser notifications
- **Notification Frequency** - Real-time or digest
- **Default Preferences** - User defaults

## Channel Settings

### Channel Configuration

- **Allow Channel Creation** - Who can create channels
- **Default Channels** - Auto-join channels
- **Channel Moderation** - Moderation requirements
- **Private Channels** - Enable private channels

## Course Settings

### Course Configuration

- **Enable Courses** - Turn on/off courses
- **Course Creation** - Who can create courses
- **Certificates** - Enable completion certificates

## Saving Settings

### Applying Changes

After modifying settings:
1. Review your changes
2. Click **Save** or **Update Settings**
3. Changes apply immediately
4. Some settings may require restart

## Best Practices

### Settings Management

- **Document changes** - Keep track of modifications
- **Test thoroughly** - Test in staging first
- **Backup settings** - Export configuration
- **Review regularly** - Audit settings periodically
- **Secure credentials** - Protect sensitive data

### Security Settings

- **Strong passwords** - Enforce requirements
- **Enable 2FA** - If available
- **Limit access** - Restrict admin settings
- **Regular audits** - Review security settings
- **Update regularly** - Keep platform updated

## Troubleshooting

### Settings Not Saving

If settings won't save:
1. Check internet connection
2. Verify admin permissions
3. Check for validation errors
4. Clear browser cache
5. Try different browser

### Changes Not Applying

If changes don't take effect:
1. Refresh the page
2. Clear application cache
3. Restart services (if applicable)
4. Check error logs
5. Contact support
