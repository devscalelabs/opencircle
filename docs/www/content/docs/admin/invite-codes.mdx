# Invite Codes

Invite codes allow you to control registration and automatically add users to specific channels.

## Creating Invite Codes

### Generate New Code

1. Navigate to **Invite Codes** in admin dashboard
2. Click **Create Invite Code** or **New Code**
3. Configure code settings:
   - Code value (auto-generated or custom)
   - Usage limit (number of uses)
   - Expiration date
   - Auto-join channels
4. Save code

## Invite Code Settings

### Configuration Options

Set invite code parameters:
- **Code** - Unique identifier
- **Max Uses** - Limit number of registrations
- **Expiration** - Set expiry date/time
- **Channels** - Auto-join specific channels
- **Description** - Internal note about code purpose

### Invite Code List

### Editing Codes

Modify existing codes:
1. Select code from list
2. Click **Edit**
3. Update settings
4. Save changes

### Deactivating Codes

Disable invite codes:
1. Select code
2. Click **Deactivate** or **Disable**
3. Code can no longer be used

### Deleting Codes

Remove invite codes:
1. Select code
2. Click **Delete**
3. Confirm deletion

## Use Cases

### Controlled Registration

Use invite codes to:
- Limit platform access
- Create exclusive communities
- Manage growth rate
- Track referral sources

### Channel Auto-Join

Automatically add users to channels:
- Onboard to specific communities
- Organize by interest groups
- Create cohorts
- Segment user base

### Campaign Tracking

Track registration campaigns:
- Different codes for different campaigns
- Monitor conversion rates
- Measure marketing effectiveness

## Best Practices

### Code Management

- **Descriptive names** - Note code purpose
- **Set limits** - Prevent abuse
- **Monitor usage** - Track effectiveness
- **Regular cleanup** - Remove expired codes
- **Secure codes** - Don't share publicly unless intended
