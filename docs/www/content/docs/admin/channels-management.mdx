# Channel Management

Learn how to create, configure, and moderate channels on your OpenCircle instance.

## Creating Channels

### Channel Creation

1. Navigate to **Channels** in admin dashboard
2. Click **Create Channel**
3. Fill in details:
   - Channel name
   - Description
   - Privacy settings
   - Channel rules
4. Save channel

## Channel Settings

### Configuration Options

Configure channel settings:
- **Name** - Channel identifier
- **Description** - What the channel is about
- **Privacy** - Public or private
- **Auto-join** - Add new users automatically
- **Moderation** - Enable/disable moderation

## Managing Channels

### Channel List

View all channels:
- Channel names

- Activity levels
- Quick actions

**IMAGE PLACEHOLDER: Channels list**
*Description: Screenshot of channels management interface*

### Editing Channels

Modify existing channels:
1. Select channel from list
2. Click **Edit**
3. Update settings
4. Save changes

### Deleting Channels

Remove channels:
1. Select channel
2. Click **Delete**
3. Confirm deletion
4. Channel and content removed

**Warning:** This action may be permanent.
