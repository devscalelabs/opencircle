# User Management

Comprehensive guide to managing users on your OpenCircle instance, including user accounts, roles, permissions, and moderation.

## User Overview

### Accessing User Management

1. Navigate to the Admin Dashboard
2. Click on **Users** in the navigation
3. View the user management interface

## Managing Individual Users

### Viewing User Details

To view a user's profile:
1. Click on the username in the user list
2. View detailed user information:
   - Profile information
   - Activity history
   - Content created
   - Courses enrolled
   - Channels joined

**IMAGE PLACEHOLDER: User detail page**
*Description: Screenshot of detailed user profile in admin view*

### Editing User Information

To edit a user:
1. Navigate to the user's detail page
2. Click **Edit** or **Edit User**
3. Modify:
   - Username (if allowed)
   - Full name
   - Email address
   - Role
   - Status
4. Save changes

**IMAGE PLACEHOLDER: Edit user form**
*Description: Screenshot of user editing interface*

### User Roles

Available roles:
- **Admin** - Full administrative access
- **User** - Standard user permissions
- **Moderator** - Limited moderation capabilities (if supported)

**Changing User Role:**
1. Open user edit page
2. Select new role from dropdown
3. Save changes
4. User receives new permissions immediately

### Deactivating Users

To deactivate a user account:
1. Navigate to user detail page
2. Click **Deactivate** or change status
3. Confirm action
4. User can no longer log in

**Note:** Deactivation is usually reversible.

### Deleting Users

To permanently delete a user:
1. Navigate to user detail page
2. Click **Delete User**
3. Confirm deletion
4. User and associated data are removed

**Warning:** This action is usually permanent and cannot be undone.

## User Permissions

### Permission Levels

Different roles have different permissions:

**Admin:**
- Full platform access
- User management
- Content moderation
- Settings configuration
- All features

**User:**
- Create posts and content
- Enroll in courses
- Join channels
- Comment and react
- Basic features

**Moderator (if supported):**
- Content moderation
- User warnings
- Report handling
- Limited admin features

## Best Practices

### User Management

- **Regular audits** - Review user accounts periodically
- **Quick response** - Address user issues promptly
- **Clear communication** - Explain decisions to users
- **Fair treatment** - Apply rules consistently
- **Privacy respect** - Protect user data

### Security

- **Strong passwords** - Enforce password requirements
- **Monitor suspicious activity** - Watch for unusual behavior
- **Regular reviews** - Audit admin actions
- **Limit admin access** - Only trusted users
- **Document actions** - Keep moderation records

## Troubleshooting

### Can't Edit User

If you can't edit a user:
1. Check your admin permissions
2. Verify user exists
3. Refresh the page
4. Try different browser
5. Check for system errors

### User Can't Log In

If a user reports login issues:
1. Verify account is active
2. Check email verification status
3. Reset password if needed
4. Check for bans or restrictions
5. Review error logs

### Changes Not Saving

If user changes don't save:
1. Check internet connection
2. Verify form validation
3. Check required fields
4. Clear browser cache
5. Try again
