# Interactions

Interactions are how you engage with content and connect with others on OpenCircle. This guide covers all the ways you can interact with posts, articles, and other content.

## Types of Interactions

### Reactions

Reactions are quick ways to express your feelings about content.

#### How to React

1. Find the reaction button on a post or comment
2. Click to see available reactions
3. Select an emoji
4. Your reaction is added

To change your reaction:
1. Click the reaction button again
2. Select a different emoji

### Comments

Comments allow for deeper engagement and discussions.

#### Creating Comments

To comment on a post:
1. Click the comment button or comment area
2. Type your comment
3. Optionally add:
   - Mentions (@username)
   - Emojis
   - Links
4. Click **Comment** or **Reply**

**IMAGE PLACEHOLDER: Comment form**
*Description: Screenshot of the comment creation interface*

#### Comment Features

Comments support:
- **Text** - Plain text messages
- **Mentions** - Tag other users
- **URLs** - Clickable links
- **Emojis** - Express yourself
- **Line breaks** - Format your text

**IMAGE PLACEHOLDER: Comment with features**

#### Replying to Comments

Create threaded discussions:
1. Find the comment you want to reply to
2. Click **Reply**
3. Type your response
4. Submit your reply

*Description: Screenshot showing nested comment replies*

#### Editing Comments

To edit your comment:
1. Find your comment
2. Click the edit icon (usually three dots menu)
3. Make your changes
4. Save the updated comment

**Note:** Edited comments may show an "edited" indicator.

**IMAGE PLACEHOLDER: Edit comment**
*Description: Screenshot showing the edit comment option*

#### Deleting Comments

To delete a comment:
1. Find your comment
2. Click the delete option
3. Confirm deletion
4. Comment is removed

**Warning:** Deletion is usually permanent.

### Mentions

Mentions notify specific users about content.

#### How to Mention

To mention someone:
1. Type `@` in a post or comment
2. Start typing their username
3. Select from autocomplete suggestions
4. The mention is added

**IMAGE PLACEHOLDER: Mention autocomplete**
*Description: Screenshot of the mention autocomplete dropdown*

#### When to Mention

Use mentions to:
- Ask someone a question
- Bring them into a discussion
- Share relevant content with them
- Acknowledge their contribution
- Request their input

#### Mention Etiquette

Best practices:
- Only mention when necessary
- Provide context
- Don't spam mentions
- Be respectful
- Use mentions purposefully

### Sharing

Share content with others (if supported).

#### How to Share

1. Find the share button on content
2. Choose sharing method:
   - Copy link
   - Share to timeline
   - Share to channel
   - External sharing (if enabled)
3. Add your commentary (optional)
4. Share

## Interaction Best Practices

### Meaningful Engagement

Create quality interactions:
- **Be thoughtful** - Add value with your comments
- **Be respectful** - Treat others kindly
- **Be constructive** - Provide helpful feedback
- **Be relevant** - Stay on topic
- **Be genuine** - Authentic engagement builds community

### Comment Quality

Good comments:
- Add to the discussion
- Ask thoughtful questions
- Share relevant experiences
- Provide helpful information
- Show appreciation

Avoid:
- One-word responses
- Off-topic comments
- Spam or self-promotion
- Negative or disrespectful remarks
- Duplicate comments

### Reaction Usage

Use reactions to:
- Show quick appreciation
- Acknowledge content
- Express agreement
- Celebrate achievements
- Indicate understanding

Don't:
- Spam reactions
- Use reactions inappropriately
- React negatively to harass

## Interaction Etiquette

### Community Guidelines

Follow these principles:
- **Respect** - Be kind and courteous
- **Relevance** - Stay on topic
- **Value** - Add meaningful contributions
- **Honesty** - Be authentic and genuine
- **Responsibility** - Own your words

### Handling Disagreements

When you disagree:
- Be respectful
- Focus on ideas, not people
- Provide constructive feedback
- Agree to disagree if necessary
- Report harassment or abuse

### Reporting Inappropriate Interactions

If you see inappropriate content:
1. Use the report function
2. Provide details
3. Let moderators handle it
4. Don't engage with trolls

## Troubleshooting

### Can't React to Content

If reactions don't work:
- Ensure you're logged in
- Check if reactions are enabled
- Refresh the page
- Try a different browser
- Clear browser cache

### Comments Not Posting

If comments fail to post:
- Check internet connection
- Verify comment isn't empty
- Ensure you have permission
- Try refreshing the page
- Check character limits

### Mentions Not Working

If mentions don't work:
- Verify username spelling
- Check if user exists
- Ensure @ symbol is used
- Try selecting from autocomplete
- Refresh and try again

### Notifications Not Received

If you don't get interaction notifications:
- Check notification settings
- Verify notifications are enabled
- Check browser permissions
- Review email settings (if applicable)
- Contact support if issues persist
