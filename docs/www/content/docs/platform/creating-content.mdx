# Creating Content

OpenCircle provides multiple ways to share your knowledge and engage with the community. This guide covers all content creation features.

## Creating Posts

Posts are the primary way to share quick updates, thoughts, and engage in discussions.

### How to Create a Post

1. Navigate to your **Timeline**
2. Find the post creation form at the top
3. Type your message in the text area
4. Optionally add:
   - Media (images, videos)
   - URLs (automatic preview generation)
   - Mentions (@username)
   - Channel tags
5. Click **Post** to publish

### Post Features

#### Text Formatting

Posts support:
- Plain text
- Line breaks
- Emojis
- URLs (automatically converted to clickable links)

#### Mentioning Users

To mention someone:
1. Type `@` followed by their username
2. Select from the autocomplete suggestions
3. The mentioned user will receive a notification

**IMAGE PLACEHOLDER: Mention autocomplete**
*Description: Screenshot showing the mention autocomplete dropdown*

#### URL Previews

When you include a URL in your post:
- OpenCircle automatically generates a preview
- Shows title, description, and image from the linked page
- Makes your posts more engaging and informative

**IMAGE PLACEHOLDER: URL preview in post**
*Description: Screenshot of a post with an embedded URL preview card*

#### Adding Media

To add images or videos:
1. Click the media upload button
2. Select files from your device
3. Preview appears before posting
4. Supported formats: JPG, PNG, GIF, MP4

**IMAGE PLACEHOLDER: Media upload interface**
*Description: Screenshot showing media attachment in post creation*

### Channel-Specific Posts

To post in a specific channel:
1. Navigate to the channel
2. Create your post from within the channel
3. Or select the channel from the post creation form

Posts in channels are visible to all channel members.

## Sharing Resources

Resources are files, links, and materials you want to share with the community.

### How to Share a Resource

1. Switch to **Resources** mode on the Timeline
2. Click **Share Resource** or similar button
3. Add:
   - Resource title
   - Description
   - URL or file upload
   - Channel (optional)
4. Click **Share**

**IMAGE PLACEHOLDER: Resource sharing form**
*Description: Screenshot of the resource creation interface*

### Types of Resources

You can share:
- **Documents** - PDFs, presentations, documents
- **Links** - External websites, articles, tools
- **Media** - Images, videos, audio files
- **Code** - GitHub repositories, code snippets
- **Other files** - Any relevant learning materials

### Resource Best Practices

- **Clear titles** - Make it easy to understand what the resource is
- **Detailed descriptions** - Explain why it's valuable
- **Proper categorization** - Use appropriate channels
- **Quality over quantity** - Share valuable, relevant resources

## Writing Articles

Articles are long-form content pieces for in-depth knowledge sharing.

### Creating an Article

**Note:** Article creation may be restricted to certain users or roles. Check with your administrator.

If you have access:
1. Navigate to the **Articles** section
2. Click **Create Article** or **New Article**
3. Fill in:
   - Title
   - Content (with rich text editor)
   - Cover image (optional)
   - Tags or categories
4. Save as draft or publish immediately

**IMAGE PLACEHOLDER: Article editor**
*Description: Screenshot of the article creation interface with rich text editor*

### Article Editor Features

The article editor typically supports:

#### Rich Text Formatting

- **Headings** - H1, H2, H3 for structure
- **Bold, Italic, Underline** - Text emphasis
- **Lists** - Bulleted and numbered lists
- **Links** - Hyperlinks to external resources
- **Code blocks** - Syntax-highlighted code
- **Quotes** - Block quotes for emphasis
- **Images** - Inline images

**IMAGE PLACEHOLDER: Rich text formatting toolbar**
*Description: Screenshot of the article editor toolbar showing formatting options*

#### Markdown Support

Many instances support Markdown for easier formatting:

```markdown
# Heading 1
## Heading 2

**Bold text**
*Italic text*

- List item 1
- List item 2

[Link text](https://example.com)

`inline code`

\`\`\`javascript
// Code block
console.log('Hello World');
\`\`\`
```

### Article Structure

A well-structured article includes:

1. **Compelling Title** - Clear and descriptive
2. **Introduction** - Hook readers and explain what they'll learn
3. **Body** - Main content with clear sections
4. **Conclusion** - Summary and key takeaways
5. **Call to Action** - Encourage engagement

### Publishing Options

Before publishing:
- **Preview** - See how your article will look
- **Save Draft** - Save for later editing
- **Publish** - Make it live immediately
- **Schedule** - Publish at a specific time (if supported)

## Content Guidelines

### Quality Content

Create content that:
- Adds value to the community
- Is accurate and well-researched
- Is original or properly attributed
- Is relevant to your audience

### Formatting Tips

- **Use headings** to break up long content
- **Add images** to illustrate points
- **Keep paragraphs short** for readability
- **Use lists** for easy scanning
- **Include examples** to clarify concepts

### Engagement Tips

- **Ask questions** to encourage discussion
- **Respond to comments** to build community
- **Share personal experiences** to add authenticity
- **Use relevant hashtags or tags** for discoverability

## Editing and Deleting Content

### Editing Posts

To edit a post you've created:
1. Find the post on your timeline or profile
2. Click the edit icon (usually three dots menu)
3. Make your changes
4. Save the updated post

**Note:** Edited posts may show an "edited" indicator.

**IMAGE PLACEHOLDER: Post edit menu**
*Description: Screenshot showing the edit option in the post menu*

### Deleting Content

To delete content:
1. Find the post, resource, or article
2. Click the delete option (usually in the menu)
3. Confirm deletion
4. Content is permanently removed

**Warning:** Deletion is usually permanent and cannot be undone.

## Media Management

### Uploading Media

When uploading images or videos:
- **File size limits** - Check your instance limits (usually 5-10MB)
- **Supported formats** - JPG, PNG, GIF for images; MP4 for videos
- **Quality** - Use high-quality but optimized files

### Media Best Practices

- **Compress images** before uploading
- **Use descriptive filenames**
- **Add alt text** for accessibility (if supported)
- **Respect copyright** - only upload media you have rights to use

## Content Visibility

### Public vs Channel Content

- **Public posts** - Visible to all users on the platform
- **Channel posts** - Visible to channel members
- **Private content** - May not be available depending on instance

### Who Can See Your Content?

By default:
- All registered users can see public posts
- Only channel members see channel-specific content
- Your profile displays your public activity

## Troubleshooting

### Post Not Publishing

If your post won't publish:
- Check your internet connection
- Ensure you're not exceeding character limits
- Verify media files are within size limits
- Try refreshing the page

### Media Upload Failing

If media won't upload:
- Check file size and format
- Try a different file
- Clear browser cache
- Use a different browser

### Content Not Appearing

If your content doesn't appear after posting:
- Refresh the page
- Check if you're in the right channel/section
- Verify the post was successfully created
- Contact support if issues persist
