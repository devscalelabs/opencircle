# Courses

OpenCircle's course system provides structured learning experiences. This guide covers everything you need to know about taking courses on the platform.

## Browsing Courses

### Course Catalog

Access the course catalog:
1. Navigate to the **Courses** section from the main menu
2. Browse available courses
3. View course cards with:
   - Course title
   - Description
   - Creator information
   - Enrollment status

### Course Details

Click on any course to view:
- Full description
- Course curriculum
- Number of lessons
- Sections and structure
- Creator profile
- Enrollment button

## Enrolling in Courses

### How to Enroll

1. Navigate to the course detail page
2. Click **Enroll** or **Join Course**
3. Confirm enrollment
4. Access to course content is granted immediately

**IMAGE PLACEHOLDER: Enrollment button**
*Description: Screenshot highlighting the course enrollment button*

### Enrollment Status

After enrolling:
- Course appears in "My Courses" or "Enrolled Courses"
- You receive access to all course materials
- Progress tracking begins
- You may receive notifications about updates

## Taking a Course

### Accessing Course Content

1. Go to your enrolled courses
2. Click on the course you want to study
3. Navigate to the lesson you want to complete
4. Content loads in the lesson viewer

**IMAGE PLACEHOLDER: Lesson viewer**
*Description: Screenshot of the main lesson viewing interface*

### Lesson Navigation

Navigate between lessons:
- **Next** button - Move to the next lesson
- **Previous** button - Go back to previous lesson
- **Sidebar** - Jump to any lesson (if unlocked)
- **Progress indicator** - See your completion status

**IMAGE PLACEHOLDER: Lesson navigation**
*Description: Screenshot showing lesson navigation controls*

### Marking Lessons Complete

To mark a lesson as complete:
1. Finish the lesson content
2. Click **Mark as Complete** or **Complete Lesson**
3. Progress is saved automatically
4. Move to the next lesson

Some lessons may auto-complete when you:
- Finish watching a video
- Complete a quiz
- Submit an assignment

## Tips for Success

### Effective Learning

- **Set a schedule** - Dedicate regular time to learning
- **Take notes** - Write down key concepts
- **Practice** - Apply what you learn
- **Engage** - Participate in discussions
- **Review** - Revisit difficult concepts

### Staying Motivated

- **Set goals** - Define what you want to achieve
- **Track progress** - Celebrate small wins
- **Join discussions** - Connect with other learners
- **Apply knowledge** - Use what you learn in real projects

### Getting Help

If you're stuck:
- Review the lesson again
- Check course resources
- Ask in course discussions
- Contact the course creator
- Reach out to the community

## Troubleshooting

### Can't Access a Lesson

If a lesson won't load:
- Check your internet connection
- Refresh the page
- Clear browser cache
- Try a different browser
- Ensure you're enrolled in the course

### Progress Not Saving

If your progress isn't saving:
- Ensure you clicked "Mark as Complete"
- Check your internet connection
- Try logging out and back in
- Contact support if issues persist

### Video Not Playing

If videos won't play:
- Check your internet speed
- Try a different browser
- Disable browser extensions
- Update your browser
- Check if video format is supported
